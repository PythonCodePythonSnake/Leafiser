TO OPERATE THE PROGRAM DO THE FOLLOWING:
1. Open testing->images and put any number of test images you want.
2. Run test.py
3. Your results will be saved the result.txt file in testing folder.

TO ADD ANOTHER DISEASE:
1. Go to datset
2. Create new folder and add samples(~500)
3. Delete model folder in home directory
4. Run test.py

NOTE:- The directory "dataset" can be deleted after model is made to save time and storage. To do that run "remove_dataset.py"
